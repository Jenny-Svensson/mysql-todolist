# Mysql-todolist
- Easy CRUD todolist using MySQL, Nodejs and Express. 
- Using PhpMyAdmin as GUI.
- Focus with this project was to understand CRUD methods
#
- Backend: Nodejs, Express, Cors, Nodemon
- Frontend: HTML/CSS, Vite, JavaScript
